# Drift Miner
A modern Android remake of the classic "Crystal Quest," written in Kotlin.

## Current State (Prototype)
- **Physics:** Inertial drift implemented (ShipPhysics.kt).
- **Controls:** Virtual joystick (drag anywhere).
- **Objective:** Collect all magenta crystals to open the Green Gateway.
- **Enemies:** "Chaser" enemy AI implemented.
- **Loop:** Endless level progression (Level 1 -> Level 2 -> etc).

## Next Steps
